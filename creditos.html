Créditos

Una página que debería tener todas las web y programas mostrando el reconocimiento de las personas que liberan su código, dibujos, etc. A la comunidad. Aquí muestro todo lo que he usado de terceros para montar esta web.

Sin el software libre nada de esto hubiese sido posible. Por todos ellos muchas gracias.

  • Todos las ilustraciones principales de la web corresponden a undraw
  • Las ilustraciones de las páginas de error corresponden a Freepik Stories
  • Por parte del css uso Bootstrap
  • Para no empezar desde cero utilicé la plantilla de 3rdwave themes
  • Para iconos utilizo Font Awesome
  • Para añadir iconos personalizados uso la herramienta IcoMoon
  • En los bloques de código en el blog utilizo highlight.js
  • Las animaciones son ejecutadas por sal.js
